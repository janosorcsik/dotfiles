# dotfiles

My macOS dotfiles

## Installation

### Clone the git repository

```sh
git clone --recursive https://github.com/janosorcsik/dotfiles.git ~/Developer/dotfiles
```

### Run the installer

```sh
sh ~/Developer/dotfiles/install.sh
```
